### Author: Jessica Johnson

## Project Description
Abducted by Earthlings is a Python text-based adventure game built to demonstrate core programming concepts including control flow, loops, conditional logic, dictionaries, and user input handling. The player navigates through a series of rooms inside a secret alien facility, collecting items and making decisions that affect the outcome of the game.

This project represents my first full program written from scratch and demonstrates my foundational understanding of Python and interactive program design.

##Technologies Used
Python
PyCharm

## Main Concepts
- Dictionaries created to store room connections
- Decision branching
- Loops
- User input and validation

## How to Play
- Use the Game Map to help easily navigate from room to room
- Follow the onscreen instruction once the program is running
- Win by collecting the items and then battling the scientist.

## Running the Game
- Run the program from the terminal: python TextBasedGame.py
